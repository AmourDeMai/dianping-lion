# Lion 使用文档

## Lion 简介

Lion 是一个配置管理平台，可以实时推送配置变更。

## Lion 使用
### 引入 Maven 依赖
```xml
<dependency>
     <groupId>com.dianping.lion</groupId>
     <artifactId>lion-client</artifactId>
     <version>0.5.0</version>
</dependency>
```
